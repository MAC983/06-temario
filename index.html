Curso de Desarrollo Frontend

El desarrollador front end se dedica a desarrollar el código que permite mostrar la interfaz visual (User Interface –UI– o interfaz de usuario) de una página y los elementos con los que el usuario interactúa. También se encarga de hacer que la parte visible de la página sea funcional, intuitiva y ergonómica para los visitantes. De acuerdo con el plan y hoja de ruta diseñados para el desarrollo de una página web o aplicación, debe crear y mantener los elementos visuales interactivos, a la vez que mejora constantemente la página para aumentar su rendimiento y hacerla más atractiva.

  2. Introducción a CSS.
  3. Introducción a Sass
  4. Introducción a JavaScript
  5. Introducción a React.
